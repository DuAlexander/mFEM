Welcome to mFEM.

mFEM is an object-oriented, open-source finite element library that includes automatic assembly routines. The project was started to serve as a means for methods testing and comparison against advanced, massively parallel frameworks such as libMesh.

This project is currently under development, although most of the code and documentation are complete. I am currently working on publishing an article about mFEM, so the code will likely change as I write and test the various components.

Installation

Installation is trival, since mFEM relies soley on MATLAB's built in functionality. To install mFEM:

  • Download the zipped source code: mFEM zipped source .
  • Unzipped the source code in a directory of your choice (e.g., ../MATLAB/mFEM).
  • Start MATLAB and naviate to the directory of the mFEM source code.
  • Run the install function from the MATLAB command line. 
     >> install;
The install functions simply adds the necessary directories to the path and adds the documentation to the help browser. Once installed an mFEM section should be added to MATLAB's help, which may be found by opening the help browser and searching for mFEM.

Demo Problem

The following code solves a simple displacement problem using 17 lines of code, with a few extra for plotting the results. This is demo was extracted from Example 7 of the mFEM source code, which contains dozens of example problems, with and without the automatic assembly functionality.

Import the mFEM code. 

 import mFEM.*;

Generate a rectangular, 2D grid. This grid rangs from 0 to 10 in the x-direction and -0.5 to 0.5 in the y-direction. It is discritized into 30 and 20 elements in the x- and y-direction, respectively. 

mesh = FEmesh('Element','Quad4','Space','vector');
mesh.grid(0,10,-0.5,0.5,30,20);
mesh.init();

Add boundary tags to the left and right side. 

mesh.add_boundary(1, 'right'); 
mesh.add_boundary(2, 'left');

Add the finite element equations. 

sys = System(mesh);
sys.add_constant('E', 1e7, 'v', 0.3, 'P', [0;100]);
sys.add_constant('D', 'E / (1-v^2) * [1, v, 0; v, 1, 0; 0, 0, (1-v)/2]');
sys.add_matrix('K', 'B''*D*B');
sys.add_vector('f', 'N''*P', 'Boundary', 2);

Assemble the stiffness matrix and force vector. 

K = sys.assemble('K');
f = sys.assemble('f');

Extract and set the prescribed values for the essential boundary conditions and solve for the non-essential unknowns. 

ess = mesh.get_dof('Boundary', 1);
u = zeros(mesh.n_dof,1);            
u(ess) = 0;     
u(~ess) = K(~ess,~ess)\(f(~ess) - K(ess,~ess)'*u(ess));

Plot the results. 

mesh.plot(u,'-deform','Patch',{'EdgeColor','k'},'Colorbar','Displacement');
xlabel('x'); ylabel('y');
xlim([-0.1,10]);
